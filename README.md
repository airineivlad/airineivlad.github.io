# airineivlad.github.io

![Web App](img/website.png)


Based Freelancer Bootstrap Jekyll theme 

## Run Local
```
jekyll serve
```

## Dependecies

Ruby and Jekyll

 1. Ruby on Windows: https://rubyinstaller.org/downloads/ 
 2. Install Bundler and Jekyll
    ```
        gem install bundler
        gem install bundler jekyll
    ```

## How to use
 - Place a image in `/img/portfolio/`
 - Creating posts to display your projects. Use the follow as an example:
```txt
---
layout: default
modal-id: 1
date: 2020-01-18
img: cabin.png
alt: image-alt
project-date: January 2020
client: The Client
category: Web Development
description: The description of the project

---
```
